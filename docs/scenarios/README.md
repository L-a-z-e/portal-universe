# Scenarios

업무 시나리오 문서 인덱스입니다.

## 목적

시나리오 문서는 **"누가, 언제, 어떻게, 왜"**를 정의하여 기능 구현의 맥락을 명확히 합니다.

- **Actor**: 누가 이 기능을 사용하는가?
- **Trigger**: 언제 이 기능이 실행되는가?
- **Flow**: 어떻게 동작하는가?
- **Goal**: 왜 이 기능이 필요한가?
